ChatGPT Assistant for IntelliJ IDEA

This plugin makes ChatGPT request for you and paste the answers directly into the code. No needs to copy code to the browser and back, just ask GPT from the IDE and see the answer in a moment.

How It Works:

  • Go to the OpenAI site and create API key: https://platform.openai.com/api-keys.
  • In IDE open the Tools menu, select AskGPT and then select AskGpt Settings. Paste the ChatGPT API key into the GPT Token field (the first field in the form displayed).

To call the plugin just select the menue Tools -> Generate... From there select wanted AskGPT> option. Instead of using the menue, you can press Alt+Insert shortcut to get to the AskGPT options. The shortcut has the advantage of not changing current selection.

The AskGPT options will allow you to select a task, and to define what information to enclose:

  • nothing
  • a selection in current window
  • a file
  • many files

The plugin then communicates with ChatGPT and provides you with actionable solutions or suggestions, right where you need them.

Key Features:

  • Custom Asks: Query ChatGPT directly to generate custom code snippets, algorithms, or tackle programming challenges right within your IDE.
  • Refactor: Automatically generates a refactored version of the provided code. Simply specify the target(s), and let ChatGPT Assistant create comprehensive tests to ensure reliability.
  • Write Tests: Automatically generate test cases for your code. Simply specify the target(s), and let ChatGPT Assistant create comprehensive tests to ensure reliability.
  • Find Bugs: Describe your bug to ChatGPT Assistant, and it will help identify potential flaws in your code, suggest debugging steps, or offer direct solutions.
  • Create Docs: Generate documentation effortlessly. ChatGPT Assistant can document your functions, classes, and modules, making your codebase easier to navigate and maintain.
